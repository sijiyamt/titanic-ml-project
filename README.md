# Titanic Survival Prediction 🚢

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## 📊 Model Used
- Logistic Regression
- Accuracy: ~80%

## 🔧 Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## 📁 Files
- notebook.ipynb → Full ML workflow
- submission.csv → Predictions output

## 🚀 Result
Successfully built a machine learning model to predict survival based on passenger data.

## 💡 Key Insights
- Females had higher survival rate
- Higher class passengers survived more
- Age also influenced survival
